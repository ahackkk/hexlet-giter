git log - показывает список всех выполненных коммитов, которые отсортированны по дате добавления.
git log -p - сразу выводит диф для каждого коммита.
git show - можно посмотреть все изменения, сделанные в рамках одного коммита. надо написать git show  а далее, идентификатор коммита.
git blame - можно увидеть , кто последним менял конкретную строчку в файле. После git blame надо указать путь до файла.
git grep - ищет совпадение с указанной строкой во всех файлах проекта. После git grep нужно написать слово. В итоге выведутся совпадения с указанной строкой из всех файлов проекта.
